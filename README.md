# 🌐 DevConnect — Connect. Collaborate. Code.

**DevConnect** is a web application built using Django and PostgreSQL, designed to help developers connect, showcase their skills, and collaborate on meaningful projects.

I'm building this project from scratch as part of my journey to become a full-stack developer. My goal is to learn by doing — understanding how real-world applications work while building something I can proudly showcase in my portfolio and to future employers.

---

## ✨ Project Highlights

- ✅ Django project fully set up and running
- ✅ PostgreSQL database connected
- ✅ Models and admin interface (superuser) created
- ✅ Initial views and HTML templates working
- ✅ Code structured with best practices in mind
- ✅ Git for version control
